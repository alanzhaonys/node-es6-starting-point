# node-es6-starting-point [![Build Status][travis-image]][travis-url] [![Coveralls Status][coveralls-image]][coveralls-url]

Prerequisites
- Run `yarn install`

To lint
- Run `yarn lint`

To test
- Run `yarn test`

To generate test coverage report
- Run `yarn cover`

To build
- Run `yarn build`

To run
- Run `yarn start`

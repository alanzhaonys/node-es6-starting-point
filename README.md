# node-es6-starting-point [![Build Status](https://travis-ci.org/alanzhaonys/node-es6-starting-point.svg?branch=master)](https://travis-ci.org/alanzhaonys/node-es6-starting-point)

Prerequisites
- Run `yarn install`

To lint
- Run `yarn lint`

To test
- Run `yarn test`

To generate test coverage report
- Run `yarn cover`

To build
- Run `yarn build`

To run
- Run `yarn start`
